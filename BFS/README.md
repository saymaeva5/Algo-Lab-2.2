This repository contains my implementations of core graph algorithms completed during the Algorithm Laboratory course. Each section includes solutions to real online judge problems using BFS, Dijkstra, and Bellman–Ford techniques.

# Lab Report 1 — BFS Problems
Algorithms covered: Breadth-First Search, bipartite checking, shortest path in unweighted graphs, and BFS on grids.

Bicoloring — Bipartite detection using BFS

Risk — Shortest path in an unweighted graph

Knight Moves — Minimum knight moves on a 2D chessboard grid
# Lab Report 2 — Dijkstra Problems
Focuses on Dijkstra’s Algorithm for weighted shortest paths and extended variants.

Dijkstra? (Codeforces 20C) — Classical shortest-path computation

Not the Best (LightOJ) — Finds the second shortest path in a weighted graph
# Lab Report 3 — Bellman–Ford Problems
Explores negative weights, cycle detection, and distance computation using the Bellman–Ford Algorithm.

UVA 558 — Wormholes — Negative cycle detection

LOJ 1108 — Traffic — Minimum and maximum distances with constraints
